Project Information

Database Schema

This e-commerce platform is built on a relational database with tables for:

  • Users - Stores customer information
  • Products - Contains product details and inventory
  • Orders - Tracks customer orders
  • OrderItems - Links products to orders with quantities
  • Shipments - Contains shipping information
  • Reviews - Stores customer product reviews
  • Wishlist - Manages user wishlist items

Technical Implementation

These web access modules demonstrate database connectivity through:

  • HTML/CSS/JavaScript for frontend implementation
  • Bootstrap 5 for responsive design and UI components
  • Database Connection simulated through JavaScript
  • Stored Procedures for encapsulating database operations

Note: This frontend demonstration simulates database interactions for demonstration purposes. In a production environment, these would connect to an actual database server.
